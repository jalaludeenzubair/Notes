How an Agent Uses Memory (Step by Step)

- Observe the environment → e.g., the robot sees a wall.
- Retrieve relevant memory → look up past experiences (episodic) or rules (semantic).
- Decide an action → choose the best action based on memory.
- Update memory → store new experiences or refine knowledge.

Episodic → remembers specific experiences for decision-making.

Semantic → generalizes knowledge from multiple experiences.
